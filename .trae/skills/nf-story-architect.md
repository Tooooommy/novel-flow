# 故事架构技能

## 使用场景

- 用户要生成小说大纲
- 用户要设计世界观设定
- 用户要创建人物设定
- 用户要生成分卷大纲

---

## 命令

### full - 生成完整大纲

```
/nf-story-architect full [--genre <题材>] [--chapters <数量>] [--volumes <卷数>]
```

| 参数     | 类型   | 必填 | 默认值   | 说明       |
| -------- | ------ | ---- | -------- | ---------- |
| genre    | string | 否   | 当前项目 | 题材类型   |
| chapters | number | 否   | 30       | 每卷章节数 |
| volumes  | number | 否   | 3        | 分卷数量   |

**执行流程**:

```
┌─────────────────────────────────────────────────────────────────────┐
│                      full 命令执行流程                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  1. world (构建世界观)                                               │
│     │ → 力量体系、地理设定、势力分布                                   │
│     │                                                                │
│     └──→ 2. characters (设计人物)                                     │
│               │ → 主角、配角、反派、女主设定                           │
│               │                                                       │
│               └──→ 3. structure (篇章结构)                           │
│                         │ → 主线、副本、章节规划                        │
│                         │                                              │
│                         └──→ 4. outline (生成大纲)                    │
│                                   │ → 完整主线/分卷大纲                 │
│                                   │                                    │
│                                   └──→ 5. volumes (生成分卷)            │
│                                             │ → 分卷详细大纲            │
│                                             ↓                          │
│                                       大纲完成                          │
└─────────────────────────────────────────────────────────────────────┘
```

**执行方案**:

| 步骤          | 操作               | 产出       |
| ------------- | ------------------ | ---------- |
| 1. world      | 构建完整世界观体系 | 世界观文档 |
| 2. characters | 设计主要人物设定   | 人物档案   |
| 3. structure  | 规划篇章结构       | 结构文档   |
| 4. outline    | 生成主线剧情大纲   | 大纲文档   |
| 5. volumes    | 生成分卷大纲       | 分卷文档   |

---

## 内部命令

### world - 构建世界观

```
/nf-story-architect world [--genre <题材>] [--scale <规模>]
```

**执行方案**:

| 步骤 | 操作                   | 产出         |
| ---- | ---------------------- | ------------ |
| 1    | 确定题材类型和核心设定 | 题材定位     |
| 2    | 设计力量体系           | 力量等级文档 |
| 3    | 构建地理/空间设定      | 地图/场景    |
| 4    | 设计势力分布           | 势力关系图   |
| 5    | 制定世界规则           | 规则文档     |

**世界观构建模板**:

```
【玄幻题材示例】

1. 力量体系
   等级: 炼气 → 筑基 → 金丹 → 元婴 → 化神 → 大乘 → 渡劫
   特殊: 体质/血脉/天赋

2. 地理设定
   中州（中央大陆）
   ├─ 宗门势力
   ├─ 世家势力
   └─ 散修聚集地

3. 势力分布
   正道: 三大圣地
   魔道: 七大宗门
   中立: 万宝阁

4. 世界规则
   - 灵气浓度分布
   - 修炼资源稀缺度
   - 宗门传承方式
```

### characters - 设计人物

```
/nf-story-architect characters [--count <数量>]
```

**执行方案**:

| 步骤 | 操作                 | 产出     |
| ---- | -------------------- | -------- |
| 1    | 确定主角核心设定     | 主角定位 |
| 2    | 设计主角性格和成长线 | 成长弧   |
| 3    | 设计女主/重要配角    | 人物设定 |
| 4    | 设计反派/敌对势力    | 反派设定 |
| 5    | 确定人物关系网络     | 关系图谱 |

**人物设定模板**:

```
【主角设定】

姓名: 萧尘
身份: 青云宗外门弟子 → ... → 巅峰强者
性格: 外冷内热、重情重义、杀伐果断
外貌: （根据题材设定）

成长线:
- 起点: 被欺辱的废物
- 转折: 意外获得传承
- 中段: 快速成长、展露天赋
- 高潮: 击败最强敌人
- 结局: 登顶巅峰

金手指: 上古剑仙传承（可进化）
目标: 复仇 + 登顶武道
```

### structure - 篇章结构

```
/nf-story-architect structure [--volumes <卷数>] [--chapters <总章节>]
```

**执行方案**:

| 步骤 | 操作                 | 产出     |
| ---- | -------------------- | -------- |
| 1    | 确定总卷数和总章节数 | 结构框架 |
| 2    | 规划每卷的主题和目标 | 卷主题   |
| 3    | 设计每卷的起承转合   | 卷结构   |
| 4    | 规划章节间的衔接     | 章节关联 |
| 5    | 确定爽点/高潮分布    | 节奏规划 |

**三段式结构**:

```
第一卷【铺垫期】: 建立世界，引出主角，埋下伏笔
  - 前5章: 困境/觉醒
  - 中20章: 成长/初露锋芒
  - 后5章: 第一个小高潮

第二卷【成长期】: 格局扩大，矛盾升级
  - 前5章: 新挑战
  - 中20章: 势力扩张/感情线
  - 后5章: 重大转折

第三卷【高潮期】: 最终对决，完美收尾
  - 前5章: 最终boss出现
  - 中20章: 连环高潮
  - 后5章: 圆满结局
```

### outline - 生成大纲

```
/nf-story-architect outline [--framework <框架>] [--theme <主题>]
```

**执行方案**:

| 步骤 | 操作             | 产出     |
| ---- | ---------------- | -------- |
| 1    | 确定主线剧情框架 | 主线骨架 |
| 2    | 设计关键情节点   | 节点列表 |
| 3    | 规划副本/支线    | 副本设计 |
| 4    | 生成章节概要     | 章节大纲 |
| 5    | 整理成完整文档   | 大纲文档 |

**大纲格式**:

```markdown
# 总大纲

## 主线

1. 起源（1-10章）
   - 萧家废物少主
   - 家族大比受辱
   - 偶得剑仙传承

2. 崛起（11-30章）
   ...

## 副本

- 妖兽森林历练
- 宗门大比
- ...

## 伏笔

- 神秘势力的出现
- 萧尘身世之谜
- ...
```

### volumes - 生成分卷

```
/nf-story-architect volumes [--count <卷数>]
```

**执行方案**:

| 步骤 | 操作         | 产出      |
| ---- | ------------ | --------- |
| 1    | 确定分卷主题 | 卷主题    |
| 2    | 设计分卷主线 | 卷主线    |
| 3    | 规划分卷章节 | 章节列表  |
| 4    | 设计分卷高潮 | 爽点设计  |
| 5    | 规划卷间衔接 | 承接/转折 |

**分卷大纲模板**:

```markdown
# 第一卷：觉醒

## 卷主题

废物逆袭，初露锋芒

## 核心情节

1. 被退婚（触发事件）
2. 觉醒金手指（转折点）
3. 家族大比夺冠（第一个爽点）
4. 被宗门长老收为亲传（身份提升）

## 章节规划

- 1-3章: 困境描写
- 4-7章: 觉醒过程
- 8-10章: 小试牛刀
- 11-15章: 第一个副本
- 16-20章: 家族大比

## 卷末悬念

大比后，神秘势力登场...
```

## 输出结构

```
novels/{project}/
├── outline/
│   ├── overview.md        # 总览
│   ├── world.md          # 世界观
│   ├── characters.md     # 人物设定
│   ├── plot.md           # 主线大纲
│   ├── timeline.md       # 时间线
│   ├── volume-1.md       # 第一卷大纲
│   ├── volume-2.md       # 第二卷大纲
│   └── volume-3.md       # 第三卷大纲
```

## 错误处理

| 错误码 | 说明     | 处理方式     |
| ------ | -------- | ------------ |
| SA-001 | 题材为空 | 使用默认题材 |
| SA-002 | 卷数无效 | 默认3卷      |
| SA-003 | 章节超限 | 自动调整     |
