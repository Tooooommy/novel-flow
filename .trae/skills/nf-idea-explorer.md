# Skill: nf-idea-explorer

## 描述

创意探索师技能，为 `/nf idea` 命令提供创意生成能力。分析热门网文趋势，结合用户偏好，生成多个差异化的创意方案。

## 调用方式

由主控技能 `/nf idea` 调用，无需直接使用。

---

## full

**生成创意方案** - 供用户选择

```
/nf-idea-explorer full --count <数量> --genre <题材>
```

**参数说明**:
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| count | number | 否 | 5 | 生成创意数量 |
| genre | string | 否 | - | 题材方向 |

**执行流程**:

1. 分析热门网文趋势
2. 结合题材类型生成概念
3. 构建差异化创意方案
4. 输出结构化推荐列表

**输出示例**:

```
┌─────────────────────────────────────────────────┐
│ 创意方案                                        │
├─────────────────────────────────────────────────┤
│ [1] 都市修仙：废物逆袭，校花倒贴                │
│     标签：都市｜修仙｜热血｜爽文                 │
│     亮点：身份反差+系统流+暧昧流                │
│                                                 │
│ [2] 异世穿越：成为史上最弱召唤师               │
│     标签：奇幻｜穿越｜召唤流｜搞笑             │
│     亮点：反套路+宠物成长+轻松向               │
└─────────────────────────────────────────────────┘
```

---

## 内部子命令

### trend

**分析网文趋势**

```
/nf-idea-explorer trend --genre <题材>
```

### concept

**生成核心概念**

```
/nf-idea-explorer concept --genre <题材> --count <数量>
```

### twist

**添加创新元素**

```
/nf-idea-explorer twist --base <基础概念> --type <类型>
```

---

## 错误处理

| 错误码 | 说明     | 处理方式         |
| ------ | -------- | ---------------- |
| IE-001 | 题材为空 | 使用通用题材生成 |
| IE-002 | 数量无效 | 默认生成5个      |
