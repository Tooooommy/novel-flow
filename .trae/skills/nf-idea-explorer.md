# 创意探索技能

## 使用场景

- 用户说"我不知道写什么"
- 用户要生成多个创意方案选择
- 用户要分析当前热门网文趋势
- 用户要针对某题材获取创意

---

## 命令

### full - 生成创意方案

```
/nf-idea-explorer full [--count <数量>] [--genre <题材>]
```

| 参数  | 类型   | 必填 | 默认值 | 说明     |
| ----- | ------ | ---- | ------ | -------- |
| count | number | 否   | 5      | 创意数量 |
| genre | string | 否   | -      | 题材方向 |

**执行**: trend → concept → twist

**输出示例**:

```
[1] 都市修仙：废物逆袭，校花倒贴
    标签：都市｜修仙｜热血｜爽文
    亮点：身份反差+系统流

[2] 异世穿越：成为史上最弱召唤师
    标签：奇幻｜穿越｜召唤流
    亮点：反套路+宠物成长
```

---

## 内部命令

### trend - 分析趋势

```
/nf-idea-explorer trend [--genre <题材>]
```

### concept - 生成概念

```
/nf-idea-explorer concept [--genre <题材>] [--count <数量>]
```

### twist - 添加创新

```
/nf-idea-explorer twist [--base <基础概念>]
```

---

## 错误处理

| 错误码 | 说明     | 处理方式     |
| ------ | -------- | ------------ |
| IE-001 | 题材无效 | 使用通用题材 |
| IE-002 | 数量无效 | 默认5个      |
