# 项目初始化技能

## 使用场景

- 用户要创建新小说项目
- 用户要初始化项目目录结构
- 用户要生成项目元数据

---

## 命令

### init - 项目立项

```
/nf init --name <名称> --genre <题材> --target <字数> [--chapter-size <每章字数>]
```

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| name | string | 是 | - | 小说名称 |
| genre | string | 是 | - | 题材类型 |
| target | number | 是 | - | 目标字数 |
| chapter-size | number | 否 | 3500 | 每章字数 |

---

## 自动计算规则

| 步骤 | 计算逻辑 |
|------|----------|
| 1 | 总章节数 = target ÷ chapter-size |
| 2 | 卷数 = ceil(总章节数 ÷ 100)，每卷约100章 |
| 3 | 每卷章节数 = ceil(总章节数 ÷ 卷数) |

**计算示例**：

| 目标字数 | 每章字数 | 总章节数 | 卷数 | 每卷章数 |
|----------|----------|----------|------|----------|
| 100万 | 3500 | 286 | 3 | 96 |
| 300万 | 3500 | 858 | 9 | 95 |
| 500万 | 3500 | 1429 | 15 | 95 |
| 800万 | 3500 | 2286 | 23 | 99 |
| 1000万 | 3500 | 2857 | 29 | 99 |

## 目录结构

```
novels/{name}/
├── novel.yaml      # 项目元数据
├── outline/        # 大纲目录
│   ├── overview.md     # 总览
│   ├── world.md        # 世界观
│   ├── characters.md    # 人物
│   ├── plot.md          # 主线
│   ├── timeline.md      # 时间线
│   └── volume-*.md      # 分卷大纲
├── content/        # 正文目录
│   └── volume-1/ch-001.md
└── research/      # 研究目录
```

---

## novel.yaml 模板

```yaml
name: {小说名称}
genre: {题材类型}
target: {目标字数}
chapter_size: {每章字数}
volumes: {卷数}
chapters: {总章节数}
chapters_per_volume: {每卷章节数}
created: {创建日期}
```

## 执行流程

```
1. 验证参数
     │
     ├──→ name为空 → 提示用户输入
     ├──→ genre无效 → 使用默认题材（玄幻）
     └──→ target无效 → 提示输入有效字数

2. 计算项目参数
     │
     ├──→ 计算总章节数
     ├──→ 计算卷数
     └──→ 计算每卷章节数

3. 创建目录结构
     │
     ├──→ 创建 novels/{name}/
     ├──→ 创建 outline/ 子目录
     ├──→ 创建 content/ 子目录
     └──→ 创建 research/ 子目录

4. 生成项目文件
     │
     ├──→ 生成 novel.yaml
     └──→ 生成 README.md

5. 输出完成信息
```

## 错误处理

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| PI-001 | 名称为空 | 提示用户输入小说名称 |
| PI-002 | 题材无效 | 使用默认题材（玄幻） |
| PI-003 | 字数无效 | 提示输入有效字数（≥10万） |
| PI-004 | 目录已存在 | 询问是否覆盖或重命名 |
| PI-005 | 创建失败 | 提示权限问题或磁盘空间 |

**错误恢复流程**:

```
PI-001: 提示并要求输入小说名称
PI-002: 使用玄幻题材，继续执行
PI-003: 提示有效字数范围（10万-5000万）
PI-004: 
  - 覆盖：删除旧目录，重新创建
  - 重命名：使用新名称创建
PI-005: 检查磁盘空间和目录权限
```
