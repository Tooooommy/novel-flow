# 合规审查技能

## 使用场景

- 用户要检查敏感内容
- 用户要过滤违规词汇
- 用户要检查平台规范
- 用户要进行发布前合规审查

---

## 命令

### full - 合规审查

```
/nf-compliance-officer full [--chapter <章节号>] [--volume <卷号>]
```

| 参数    | 类型   | 必填 | 默认值 | 说明   |
| ------- | ------ | ---- | ------ | ------ |
| chapter | number | 否   | 当前   | 章节号 |
| volume  | number | 否   | 当前   | 卷号   |

**执行流程**:

1. 调用sensitive技能检查敏感内容:
   - 政治、色情、暴力、违法犯罪
   - 敏感词、违禁词、限制词
   - 人物设定、能力、背景
   - 势力设定、关系、实力、规矩
   - 道具设定、能力、限制、归属
   - 数量是否混乱
2. 调用vocabulary技能过滤违规词汇:
   - 敏感词、违禁词、限制词
3. 调用platform技能检查平台规范:
   - 各平台特殊要求
4. 输出合规审查报告

**输出**:

合规审查报告包含以下内容：

1. 是否通过审查
2. 问题列表: 包含所有违规内容的详细信息
3. 警告列表: 包含所有警告内容的详细信息
4. 修复建议: 包含所有修复建议的详细信息

## 敏感内容检测

### sensitive - 敏感内容

```
/nf-compliance-officer sensitive [--text <文本>] [--level <级别>]
```

**检测维度**:

| 维度     | 说明                   | 检测方法        | 严重程度 |
| -------- | ---------------------- | --------------- | -------- |
| 政治敏感 | 国家领导人、历史事件等 | 关键词+语义分析 | 严重     |
| 色情低俗 | 色情描写、脏话等       | 关键词+场景分析 | 严重     |
| 暴力血腥 | 过度血腥、残忍描写     | 关键词+程度分析 | 中等     |
| 违法犯罪 | 赌博、毒品、诈骗等     | 关键词+行为分析 | 严重     |
| 封建迷信 | 鬼神、邪教等           | 关键词+内容分析 | 中等     |
| 人身攻击 | 侮辱、诽谤、造谣等     | 人名+行为分析   | 中等     |

**敏感内容分级**:

| 级别 | 说明     | 处理方式 |
| ---- | -------- | -------- |
| S级  | 绝对禁止 | 必须删除 |
| A级  | 严重违规 | 必须修改 |
| B级  | 中度违规 | 建议修改 |
| C级  | 轻度违规 | 谨慎使用 |

## 违规词汇过滤

### vocabulary - 词汇

```
/nf-compliance-officer vocabulary [--text <文本>] [--platform <平台>]
```

**词汇分类**:

| 类别   | 说明     | 示例       | 处理方式 |
| ------ | -------- | ---------- | -------- |
| 违禁词 | 完全禁止 | [具体词汇] | 必须替换 |
| 敏感词 | 需要审核 | [具体词汇] | 建议替换 |
| 限制词 | 限制使用 | [具体词汇] | 控制频率 |
| 风险词 | 需要注意 | [具体词汇] | 谨慎使用 |

**违禁词库（部分）**:

```markdown
【绝对禁止词汇】

- 毒品相关: [词汇列表]
- 赌博相关: [词汇列表]
- 诈骗相关: [词汇列表]
- 邪教相关: [词汇列表]
```

**敏感词库（部分）**:

```markdown
【需要审核词汇】

- 尸体/死亡相关
- 血液/伤害相关
- 性暗示相关
- 权力相关
```

**替代词建议**:

| 违禁词 | 替代词   | 替代效果 |
| ------ | -------- | -------- |
| [原词] | [替代词] | 保持语义 |
| [原词] | [替代词] | 保持语义 |

**词汇检测规则**:

| 规则       | 说明         | 示例     |
| ---------- | ------------ | -------- |
| 精确匹配   | 完全相同的词 | "赌博"   |
| 变形匹配   | 词形变化     | "赌博的" |
| 同音匹配   | 同音不同字   | "Du博"   |
| 谐音匹配   | 近似发音     | "赌*B*"  |
| 拼音首字母 | 拼音缩写     | "DB"     |

**替换标准流程**:

```
1. 检测到违规词
     │
     ├──→ 判断级别
     │     │
     │     ├──→ S/A级 → 强制替换
     │     ├──→ B级 → 建议替换
     │     └──→ C级 → 提醒注意
     │
     └──→ 应用替换规则
           │
           ├──→ 首选替代词
           ├──→ 次选替代词
           └──→ 语义调整
```

## 错误处理

| 错误码 | 说明       | 处理方式     |
| ------ | ---------- | ------------ |
| CO-001 | 章节不存在 | 返回空结果   |
| CO-002 | 文本为空   | 返回错误     |
| CO-003 | 平台不支持 | 使用通用规范 |
| CO-004 | 审查超时   | 返回部分结果 |

**错误恢复流程**:

```
1. 检测错误类型
     │
     ├──→ CO-001 → 返回"章节不存在"
     │
     ├──→ CO-002 → 返回"文本为空"
     │
     ├──→ CO-003 → 使用通用规范继续
     │
     └──→ CO-004 → 返回已检测内容
```
