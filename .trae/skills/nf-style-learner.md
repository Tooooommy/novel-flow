# Skill: nf-style-learner

## 描述

风格学习技能，支持两种模式：

1. **用户样本学习**：用户提供文风样本，系统学习并模仿
2. **作家风格模仿**：直接选择知名网文作家风格，系统应用其写作特征

## 调用方式

```
/nf-style-learner <command> [options]
```

## 命令

### list

列出可用的写作风格（包括内置作家风格）

### 猫腻

**代表作品**: 《庆余年》《间客》《将夜》

**风格特征**:
| 维度 | 特征描述 |
|------|---------|
| 句式 | 长短交错，节奏感强，善用短句制造张力 |
| 用词 | 文雅精致，书面语与口语结合，偶用古语 |
| 修辞 | 善用意象，比喻新颖，心理描写细腻 |
| 对话 | 简短有力，潜台词丰富，有文人气质 |
| 特色 | 主角多谋深算，配角鲜明有血有肉 |

**核心写作规则**:

```yaml
- 优先使用短句制造节奏感
- 对话不超过20字，惜字如金
- 心理描写用意象展示，不直接陈述
- 环境描写服务于氛围，简洁有力
- 善用"然而""可是""只是"等转折
```

**风格Prompt**:

```
你正在模仿作家猫腻的文风：
1. 句式：长短交错，以短句为主，每段不超过3句
2. 对话：简洁有力，每句对话不超过20字，善用潜台词
3. 心理：用动作和细节展示内心，不直说"感到XX"
4. 用词：文雅精致，偶用古语词汇如"罢了""却是""颇为"
5. 段落：短段落为主，每段2-3句，营造阅读节奏
6. 特色：主角深沉多谋，配角对话常有弦外之音
```

## 去AI味道核心技巧

无论使用哪种风格，都应遵循以下去AI味道原则：

### 1. Show Don't Tell

```markdown
# ❌ AI写法

他感到非常愤怒和悲伤。

# ✅ 人类写法

他的手在发抖，指节泛白。眼眶发酸，却硬是没让那点湿意落下来。
```

### 2. 对话要"不完整"

```markdown
# ❌ AI写法

"你这样做是不对的，我希望你能好好反思一下。"

# ✅ 人类写法

"你……"他顿了顿，没说下去。
```

### 3. 省略不必要的修饰

```markdown
# ❌ AI写法

他缓缓地睁开眼睛，目光中带着一丝惊讶和好奇。

# ✅ 人类写法

他睁开眼，愣了一下。
```

### 4. 用动作代替陈述

```markdown
# ❌ AI写法

她对他的离开感到非常难过。

# ✅ 人类写法

她站在门口，看着他的背影消失在街角。手里的信纸被攥成一团。
```

## 错误处理

| 错误码 | 说明         | 处理方式                                 |
| ------ | ------------ | ---------------------------------------- |
| SL-001 | 样本字数不足 | 提示用户至少提供1000字（自定义风格学习） |
| SL-002 | 作家不存在   | 显示可用作家列表                         |
| SL-003 | 内容为空     | 提示用户提供要转换的内容                 |
| SL-004 | 风格冲突     | 提示用户同一段落不能应用多种风格         |
