# 质量评估技能

## 使用场景

- 用户要评估章节质量
- 用户要评分可读性
- 用户要评分情感表达
- 用户要评分节奏控制
- 用户要输出质量报告

---

## 命令

### 生成质量评估报告

> 检查章节质量，输出质量评估报告。

```
/nf-quality-assessor
```

**执行流程**:

1. 调用readability技能评估可读性:
   - 文字流畅度、句式复杂度、阅读体验
   - 文字质量评分
2. 调用emotion技能评估情感表达:
   - 情感真实性、情绪感染力、情感层次
   - 情感表达评分
3. 调用pacing技能评估节奏控制:
   - 节奏控制、张力曲线、爽点分布
   - 节奏控制评分
4. 调用character技能评估人物表达:
   - 人物塑造、对话自然、行为合理
   - 人物表达评分
5. 输出质量评估报告 `ch-{章节编号}-{章节名}-quality-assessment.md`

**输出**:

质量评估报告包含以下内容：

1. 是否通过审查
2. 各项评分: 可读性评分、 情感表达评分、 节奏控制评分、 人物塑造评分
3. 综合评分: 计算加权总分
4. 等级: 根据综合评分判断等级
5. 问题列表
6. 修复建议

**评分等级**:

| 等级   | 分数范围 |
| ------ | -------- |
| 优秀   | 90+      |
| 良好   | 75-89    |
| 及格   | 60-74    |
| 不及格 | <60      |

## 流程

### readability - 可读性评估

**评估标准**:

| 评估项     | 优秀(90+) | 良好(75-89) | 及格(60-74) | 不及格(<60) |
| ---------- | --------- | ----------- | ----------- | ----------- |
| 文字流畅度 | 行云流水  | 自然流畅    | 基本通顺    | 拗口        |
| 句式变化   | 长短交错  | 有变化      | 较单一      | 单一        |
| 用词精准   | 精确传神  | 准确        | 基本准确    | 用词不当    |
| 阅读体验   | 沉浸愉悦  | 舒适        | 可读        | 费力        |

**评分公式**: `R = 0.4×流畅度 + 0.2×句式 + 0.2×用词 + 0.2×体验`

**可读性检查清单**:

| 检查项   | 方法                    | 扣分标准            |
| -------- | ----------------------- | ------------------- |
| 长句过多 | 统计超过30字的句子比例  | >30%扣10分          |
| 重复用词 | 检测相同形容词/动词频率 | 同一词出现>5次扣5分 |
| 句式单调 | 检测句子开头重复率      | >20%扣10分          |
| 逻辑跳跃 | 检查前后句逻辑连接      | 每处跳跃扣5分       |

**优化建议生成标准**:

| 问题类型 | 建议模板                           |
| -------- | ---------------------------------- |
| 长句过多 | "建议将第X段的长句拆分为2-3个短句" |
| 用词重复 | "建议将第X段的'XX'替换为同义词"    |
| 句式单调 | "建议在第X段增加长短句交错"        |

---

### emotion - 情感评估

**评估标准**:

| 评估项   | 优秀(90+) | 良好(75-89) | 及格(60-74) | 不及格(<60) |
| -------- | --------- | ----------- | ----------- | ----------- |
| 情感真实 | 触动人心  | 能感受      | 牵强        | 虚假        |
| 情绪感染 | 强烈共鸣  | 有感染      | 一般        | 无感        |
| 情感层次 | 多层交织  | 有层次      | 较单一      | 单调        |
| 情绪转变 | 自然合理  | 基本合理    | 生硬        | 突兀        |

**评分公式**: `E = 0.3×真实感 + 0.3×感染力 + 0.2×层次 + 0.2×转变`

**情感表现检查表**:

| 正面表现（加分项） | 负面表现（扣分项） |
| ------------------ | ------------------ |
| 用身体反应展现情绪 | 直接陈述"他很生气" |
| 情感与环境映衬     | 情绪词堆砌         |
| 内心独白有深度     | 情感转变突兀       |
| 细节触发情感       | 情感孤立，缺乏铺垫 |

**扣分规则**:

| 问题         | 检测方法                                   | 扣分   |
| ------------ | ------------------------------------------ | ------ |
| 直接陈述情感 | 检测"感到"、"觉得"、"认为"后面直接跟情绪词 | 每处-2 |
| 情绪词堆砌   | 检测连续3个以上情绪词                      | -5     |
| 情感突兀     | 检测情感转变无过渡                         | -10    |
| 缺乏共鸣点   | 读者无法代入                               | -10    |

---

### pacing - 节奏评估

**评估标准**:

| 评估项   | 优秀(90+) | 良好(75-89) | 及格(60-74) | 不及格(<60) |
| -------- | --------- | ----------- | ----------- | ----------- |
| 张力曲线 | 起伏有致  | 有起伏      | 较平淡      | 平铺直叙    |
| 高潮设置 | 高潮迭起  | 有高潮      | 少高潮      | 无高潮      |
| 节奏变化 | 节奏鲜明  | 有变化      | 较单一      | 拖沓或仓促  |
| 爽点密度 | 密集合理  | 有爽点      | 较少        | 憋屈        |

**节奏检查标准**:

| 检查项   | 标准            | 不合格表现     |
| -------- | --------------- | -------------- |
| 章初引入 | 10%以内进入正题 | 废话过多       |
| 章末钩子 | 有悬念或承接    | 戛然而止       |
| 高潮位置 | 章中后或章末    | 高潮提前或靠后 |
| 节奏变化 | 至少3次明显起伏 | 从头到尾一样   |

**爽点密度标准**:

| 章节类型       | 最低爽点数 | 优秀爽点数 |
| -------------- | ---------- | ---------- |
| 标准章(3000字) | 2个        | 4-5个      |
| 过渡章         | 1个        | 2-3个      |
| 高潮章         | 4个        | 6+个       |
| 决战章         | 5个        | 8+个       |

**爽点类型权重**:

| 类型     | 权重 | 示例               |
| -------- | ---- | ------------------ |
| 逆转爽点 | 1.5  | 反杀、绝地翻盘     |
| 装逼爽点 | 1.4  | 打脸、嘲讽反击     |
| 升级爽点 | 1.3  | 突破、获得传承     |
| 情报爽点 | 1.2  | 真相揭露、原来如此 |
| 情感爽点 | 1.1  | 佳人倒贴、兄弟情深 |

---

### character - 人物评估

**评估标准**:

| 评估项   | 优秀(90+) | 良好(75-89) | 及格(60-74) | 不及格(<60) |
| -------- | --------- | ----------- | ----------- | ----------- |
| 性格一致 | 始终如一  | 基本一致    | 偶有偏差    | 前后矛盾    |
| 对话自然 | 符合人设  | 较自然      | 较生硬      | 千人一腔    |
| 行为合理 | 符合逻辑  | 基本合理    | 偶有牵强    | 不合理      |
| 人物辨识 | 特点鲜明  | 有特点      | 较模糊      | 无辨识度    |

**人物一致性检查**:

| 检查类型 | 检测方法               | 扣分 |
| -------- | ---------------------- | ---- |
| 性格矛盾 | 人物行为与设定性格不符 | -15  |
| 对话偏离 | 对话不符合人物背景     | -10  |
| 能力矛盾 | 前后实力表现不一致     | -10  |
| 记忆矛盾 | 对之前事件无记忆       | -5   |

**对话评分标准**:

| 评估维度 | 优秀       | 合格       | 不合格   |
| -------- | ---------- | ---------- | -------- |
| 口语化   | 自然对话   | 偶有书面语 | 过于书面 |
| 潜台词   | 有言外之意 | 偶有潜台词 | 直白无味 |
| 性格体现 | 闻声知人   | 基本可辨   | 千人一腔 |
| 信息量   | 含信息推进 | 基本推进   | 废话连篇 |

## 质量等级定义

**综合评分标准**:

| 等级 | 分数范围 | 含义         | 处理方式   |
| ---- | -------- | ------------ | ---------- |
| S级  | 95-100   | 完美无需修改 | 直接通过   |
| A级  | 90-94    | 优秀轻微瑕疵 | 小修小改   |
| B级  | 80-89    | 良好需要优化 | 针对性修改 |
| C级  | 70-79    | 及格较多问题 | 需要返修   |
| D级  | 60-69    | 不及格需重写 | 建议重写   |
| F级  | <60      | 严重问题     | 必须重写   |

**各维度等级要求**:

| 综合等级 | 可读性 | 情感 | 节奏 | 人物 | 说明         |
| -------- | ------ | ---- | ---- | ---- | ------------ |
| S        | ≥90    | ≥90  | ≥90  | ≥90  | 所有维度优秀 |
| A        | ≥85    | ≥85  | ≥85  | ≥85  | 无明显短板   |
| B        | ≥75    | ≥75  | ≥75  | ≥75  | 有提升空间   |
| C        | ≥65    | ≥65  | ≥65  | ≥65  | 问题较多     |
| D        | ≥55    | ≥55  | ≥55  | ≥55  | 需大幅修改   |
| F        | <55    | <55  | <55  | <55  | 需重写       |

**通过标准**:

| 发布场景 | 最低等级 | 必须达标维度      |
| -------- | -------- | ----------------- |
| 初稿审核 | C级(70)  | 节奏≥70           |
| 精修审核 | B级(80)  | 可读性≥75,节奏≥75 |
| 发布审核 | A级(85)  | 所有维度≥80       |
| 精品审核 | S级(95)  | 所有维度≥90       |

## 输出报告格式

**完整质量评估报告**:

```markdown
# 质量评估报告

## 基本信息

- 章节: 第X章
- 字数: XXXX字
- 评估时间: XXXX-XX-XX

## 综合评分

| 维度     | 分数   | 等级     |
| -------- | ------ | -------- |
| 可读性   | 85     | 良好     |
| 情感     | 78     | 良好     |
| 节奏     | 82     | 良好     |
| 人物     | 80     | 良好     |
| **综合** | **81** | **良好** |

## 详细分析

### 可读性评估

- 文字流畅度: ★★★★☆
- 句式变化: ★★★★☆
- 用词精准: ★★★★☆
- 阅读体验: ★★★★☆

**主要问题**: 第3段长句较多

**优化建议**:

1. 将第3段的长句拆分为2-3个短句
2. 减少"的"字结构使用

### 情感评估

- 情感真实: ★★★★☆
- 情绪感染: ★★★☆☆
- 情感层次: ★★★★☆
- 情绪转变: ★★★★☆

**主要问题**: 第5段情感转变较突兀

**优化建议**:

1. 在第4段末尾增加情感铺垫
2. 第5段增加过渡句

### 节奏评估

- 张力曲线: 起伏有致
- 高潮设置: 位置恰当
- 节奏变化: 变化合理
- 爽点密度: 达标

**主要问题**: 章初引入稍慢

**优化建议**:

1. 减少第1段的背景描写
2. 加快开篇节奏

### 人物评估

- 性格一致: ★★★★☆
- 对话自然: ★★★★☆
- 行为合理: ★★★★☆
- 人物辨识: ★★★☆☆

**主要问题**: 配角辨识度略低

**优化建议**:

1. 给配角增加标志性口头禅
2. 增加区分度的外貌描写

## 改进优先级

| 优先级 | 问题         | 维度 | 建议     |
| ------ | ------------ | ---- | -------- |
| P1     | 章初节奏慢   | 节奏 | 精简开头 |
| P2     | 情感转变突兀 | 情感 | 增加过渡 |
| P3     | 配角辨识度低 | 人物 | 增加特色 |

## 最终结论

**质量等级**: B级（良好）

**是否通过**: 是

**备注**: 本章整体质量良好，建议进行针对性优化后发布。
```

## 错误处理

| 错误码 | 说明       | 处理方式             |
| ------ | ---------- | -------------------- |
| QA-001 | 章节不存在 | 返回空结果，提示创建 |
| QA-002 | 文本为空   | 跳过评估，返回错误   |
| QA-003 | 文本过短   | 提示至少需要500字    |
| QA-004 | 格式错误   | 返回格式错误说明     |

**错误恢复流程**:

```
1. 检测错误类型
     │
     ├──→ QA-001 → 返回"章节不存在"
     │
     ├──→ QA-002 → 返回"文本为空"
     │
     ├──→ QA-003 → 返回"文本过短，建议至少500字"
     │
     └──→ QA-004 → 返回格式错误说明
```
