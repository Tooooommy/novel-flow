# 风格学习技能

## 使用场景

- 用户要模仿某作家风格
- 用户要学习自定义文风
- 用户要设置默认写作风格

## 作家详细风格分析

### 猫腻

**基本信息**:

| 项目     | 内容                                 |
| -------- | ------------------------------------ |
| 代表作   | 《庆余年》《间客》《将夜》《择天记》 |
| 擅长题材 | 权谋、穿越、历史架空、玄幻           |
| 文风定位 | 文青伏笔流                           |
| 读者评价 | "老猫的书，每一遍都有新发现"         |

**风格特征详解**:

| 类别         | 要点                                                                         |
| ------------ | ---------------------------------------------------------------------------- |
| **叙事节奏** | 慢热型(前30万字铺垫) · 张弛有度(智斗后情感释放) · 章节末尾必留钩子           |
| **文字风格** | 半文半白(对话白话，旁白文言) · 四字短句增强气势 · 心理描写细腻               |
| **人物塑造** | 主角多面(有私心有算计) · 配角落泪(每个重要配角高光) · 反派有魂(有自己的理想) |
| **情节设计** | 伏笔千里(闲笔改变全局) · 智斗为主(靠计谋非实力) · 情感深刻(爱情友情家国)     |

**写作技巧拆解**:

| 技巧类型 | 描述                       | 示例                         |
| -------- | -------------------------- | ---------------------------- |
| 氛围营造 | 用环境暗示心理             | "夜风穿过竹林，吹动他的衣袂" |
| 伏笔设置 | 看似闲笔，实则关键         | "那年的雪，下得格外大"       |
| 人物对比 | 不同人物对同一事作不同反应 | 范闲vs陈萍萍的政治手段       |
| 情绪递进 | 逐层铺垫后爆发             | 压抑→爆发→宣泄               |
| 留白艺术 | 不说破，让读者自己体会     | "你懂的"                     |

**风格样本示例**:

```原文
积善之家，必有余庆，留余庆，留余庆，忽遇黑心蛇；乘除加减，上有苍穹。

这是庆余年的开头，也是猫腻风格的典型体现。看似引经据典，实则暗藏玄机。

他写的不是简单的爽文，而是一种沉甸甸的思考。

主角不是无所不能的，他会犹豫、会软弱、会在深夜独自面对自己的内心。

但正因如此，当他最终做出选择的时候，才格外动人。
```

**适用场景**:

| 场景类型   | 适用度 | 说明                   |
| ---------- | ------ | ---------------------- |
| 权谋小说   | ★★★★★  | 智斗描写是猫腻最强项   |
| 历史穿越   | ★★★★☆  | 半文半白的调性非常适合 |
| 情感向小说 | ★★★★☆  | 细腻的心理描写加分     |
| 快节奏爽文 | ★★☆☆☆  | 猫腻的书需要耐心，慢热 |

### 辰东

**基本信息**:

| 项目     | 内容                                 |
| -------- | ------------------------------------ |
| 代表作   | 《完美世界》《遮天》《神墓》《圣墟》 |
| 擅长题材 | 东方玄幻、战斗、热血、史诗           |
| 文风定位 | 史诗热血流                           |
| 读者评价 | "东哥的书，热血到头皮发麻"           |

**风格特征详解**:

```
┌────────────────────────────────────────────────────────────────────────┐
│                        辰东风格深度解析                                   │
├────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. 世界观构建:                                                        │
│     - 宏大叙事: 诸天万界、仙古纪元、位面征战                            │
│     - 体系完整: 修炼等级分明，战力对比清晰                              │
│     - 地图宏阔: 从边荒到帝城到仙域层层递进                              │
│                                                                         │
│  2. 战斗描写:                                                          │
│     - 招式命名: 大气磅礴，如"他化自在""万劫轮回"                        │
│     - 战斗场面: 天地变色、山河破碎、星辰陨落                            │
│     - 热血对白: "谁在称无敌？""我为天帝"                               │
│                                                                         │
│  3. 爽点设计:                                                          │
│     - 越级战斗: 弱者逆袭强者，永远的爽点                                 │
│     - 打脸时刻: 曾经看不起主角的人纷纷震惊                              │
│     - 绝境翻盘: 必死之局的反转让人热血沸腾                             │
│                                                                         │
│  4. 人物塑造:                                                          │
│     - 兄弟情义: 肝胆相照，生死与共                                       │
│     - 红颜薄命: 美女角色常有好结局但过程曲折                            │
│     - 天骄云集: 配角也出彩，形成群像                                     │
│                                                                         │
└────────────────────────────────────────────────────────────────────────┘
```

**写作技巧拆解**:

| 技巧类型   | 描述                     | 示例                         |
| ---------- | ------------------------ | ---------------------------- |
| 史诗感营造 | 用天地异象衬托战斗       | "九天十地颤抖，诸天万界同悲" |
| 招式命名法 | 四字为主，气势磅礴       | "六道轮回拳""行字诀"         |
| 情绪引爆   | 压抑后的爆发最动人       | 石昊被逼入绝境后的反击       |
| 牺牲渲染   | 用配角的死推动情节和情绪 | 很多配角的牺牲催人泪下       |
| 升级快感   | 突破时的描写让人满足     | 渡劫时的天地异象             |

**风格样本示例**:

```原文
独断万古！

这是荒天帝石昊毕生的写照。

他从下界八域走出，一路征战，遇神杀神，遇佛杀佛。

没有人知道他还活着，但他活成了所有人畏惧的存在。

当那句"谁在称无敌，哪个敢言不败"响起时，整个诸天都在颤抖。

这就是辰东的风格——让每一个字都燃烧着热血。
```

**适用场景**:

| 场景类型 | 适用度 | 说明                         |
| -------- | ------ | ---------------------------- |
| 玄幻战斗 | ★★★★★  | 辰东的战斗描写是天花板级别   |
| 热血升级 | ★★★★★  | 越级战斗和打脸是他的招牌     |
| 史诗巨制 | ★★★★☆  | 宏大世界观适合超长篇         |
| 都市小白 | ★★☆☆☆  | 辰东的书普遍偏长，不适合短篇 |

### 天蚕土豆

**基本信息**:

| 项目     | 内容                                       |
| -------- | ------------------------------------------ |
| 代表作   | 《斗破苍穹》《武动乾坤》《大主宰》《元尊》 |
| 擅长题材 | 东方玄幻、升级流、逆袭流                   |
| 文风定位 | 升级爽文流                                 |
| 读者评价 | "斗破之后，再无废物流"                     |

**风格特征详解**:

```
┌────────────────────────────────────────────────────────────────────────┐
│                      天蚕土豆风格深度解析                                │
├────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. 升级体系:                                                          │
│     - 等级分明: 斗者、斗师、大斗师...每级分九星                        │
│     - 瓶颈设计: 每次突破都有明确目标和难度                             │
│     - 丹药辅助: 升级需要丹药、机缘、努力缺一不可                        │
│                                                                         │
│  2. 爽点节奏:                                                          │
│     - 321结构: 困难→努力→突破 的标准节奏                               │
│     - 打脸模式: 被轻视→忍辱负重→当众打脸                               │
│     - 越级战斗: 总是能打败比自己强的人                                  │
│                                                                         │
│  3. 套路模板:                                                          │
│     - 退婚流: 三十年河东三十年河西，莫欺少年穷                         │
│     - 拍卖会: 必定遇到挑衅，然后用钱砸死对方                           │
│     - 长老选举: 关键时刻突破，当选最年轻的长老                         │
│                                                                         │
│  4. 文字特点:                                                          │
│     - 简洁明了: 没有过多修饰，一句是一句                                │
│     - 节奏飞快: 每章都有进展，不水                                      │
│     - 画面感强: 战斗场面清晰，动作描写到位                               │
│                                                                         │
└────────────────────────────────────────────────────────────────────────┘
```

**写作技巧拆解**:

| 技巧类型   | 描述                   | 示例                       |
| ---------- | ---------------------- | -------------------------- |
| 退婚激励法 | 用屈辱激发主角斗志     | "萧炎表哥，纳兰嫣然退婚"   |
| 扮猪吃虎   | 表面废物实则天才       | 药老附体后实力暴涨         |
| 嘲讽累积   | 嘲讽越多，打脸越爽     | 一次次被轻视，最后惊艳全场 |
| 宝物助攻   | 金手指道具辅助升级     | 陨落心炎、三千雷动         |
| 境界对比   | 用具体数字展示实力差距 | "九星斗宗VS一星斗尊"       |

**风格样本示例**:

```原文
"三十年河东，三十年河西，莫欺少年穷！"

萧炎的声音在广场上回荡。

纳兰嫣然脸色微变，她没想到当年那个废物，竟敢当众说出这样的话。

"你以为说几句狠话就能改变什么？"她冷笑。

萧炎没有回答，只是缓缓抬起手掌。

掌心处，一簇青色的火焰跳动。

"这...这是异火？！"

纳兰家族的族长猛然站起，眼中满是不可置信。

从此，乌坦城再无人敢称萧炎为废物。
```

**适用场景**:

| 场景类型   | 适用度 | 说明                         |
| ---------- | ------ | ---------------------------- |
| 升级流小说 | ★★★★★  | 土豆是升级流的祖师爷         |
| 废物流逆袭 | ★★★★★  | 退婚流的开创者               |
| 东方玄幻   | ★★★★☆  | 斗气体系影响了一代作家       |
| 慢热深度文 | ★☆☆☆☆  | 土豆的书节奏飞快，不适合慢热 |

### 唐家三少

**基本信息**:

| 项目     | 内容                                     |
| -------- | ---------------------------------------- |
| 代表作   | 《斗罗大陆》《琴帝》《酒神》《神印王座》 |
| 擅长题材 | 西幻、热血、冒险、情感                   |
| 文风定位 | 简洁小白流                               |
| 读者评价 | "最适合新手入门网文的作家"               |

**风格特征详解**:

```
┌────────────────────────────────────────────────────────────────────────┐
│                      唐家三少风格深度解析                                │
├────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. 文字风格:                                                          │
│     - 通俗易懂: 没有生僻字，没有复杂句式                                │
│     - 简洁明快: 一句一段，一段一句，节奏感强                           │
│     - 直白表达: 很少用隐喻暗示，有什么说什么                            │
│                                                                         │
│  2. 设定体系:                                                          │
│     - 独创设定: 武魂、蓝银草、昊天锤...每个设定都有完整体系              │
│     - 等级清晰: 魂士、魂师、大魂师...非常容易理解                       │
│     - 魂环概念: 用颜色和年限区分，视觉化强                              │
│                                                                         │
│  3. 情感线:                                                            │
│     - 专情描写: 男主只爱一人，感情线清晰                                │
│     - 伙伴情义: 史莱克七怪的团队精神                                    │
│     - 师生情深: 大师和唐三的父子般情感                                  │
│                                                                         │
│  4. 爽点设计:                                                          │
│     - 升级明确: 每次突破都有明确展示                                     │
│     - 团队配合: 不是一个人战斗，是一群人                               │
│     - 正义必胜: 主角代表正义，邪不胜正                                   │
│                                                                         │
└────────────────────────────────────────────────────────────────────────┘
```

**写作技巧拆解**:

| 技巧类型   | 描述                       | 示例                     |
| ---------- | -------------------------- | ------------------------ |
| 武魂设定法 | 用具体形象设定角色能力     | 蓝银草觉醒为蓝银皇       |
| 团队副本   | 多人配合战斗，各有分工     | 史莱克七怪配合输出       |
| 传承羁绊   | 父母/师父的能力传承给主角  | 昊天锤的传承             |
| 颜色标识   | 用颜色区分等级/属性        | 十万年红环               |
| 简单对比   | 不用复杂计算，直接对比展示 | "他是百年魂环，我是千年" |

**风格样本示例**:

```原文
蓝银草，废武魂。

当大师说出这四个字时，唐三的心反而定了。

废武魂又如何？他还有唐门暗器。

六岁那年，他被判定为废武魂。

十二岁那年，他在史莱克学院让所有人震惊。

"我的武魂是蓝银草，"唐三站在广场中央，"但那又怎样？"

"真正强大的，从来不是武魂，而是使用它的人。"
```

**适用场景**:

| 场景类型 | 适用度 | 说明                         |
| -------- | ------ | ---------------------------- |
| 西幻冒险 | ★★★★★  | 唐家三少的西幻写得最好       |
| 新手入门 | ★★★★★  | 文笔清晰，适合学习基础节奏   |
| 团队战斗 | ★★★★☆  | 多角色配合是他的强项         |
| 复杂权谋 | ★★☆☆☆  | 三少的书普遍直白，不擅长权谋 |

### 我吃西红柿

**基本信息**:

| 项目     | 内容                                     |
| -------- | ---------------------------------------- |
| 代表作   | 《星辰变》《盘龙》《吞噬星空》《莽荒纪》 |
| 擅长题材 | 东方玄幻、星际科幻、热血                 |
| 文风定位 | 流畅热血流                               |
| 读者评价 | "番茄的书，看得最爽最流畅"               |

**风格特征详解**:

```
┌────────────────────────────────────────────────────────────────────────┐
│                      我吃西红柿风格深度解析                              │
├────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. 文字风格:                                                          │
│     - 行云流水: 读起来非常顺畅，几乎没有磕绊                           │
│     - 简洁有力: 不用华丽的辞藻，但足够动人                             │
│     - 画面感强: 战斗场面栩栩如生，让人欲罢不能                         │
│                                                                         │
│  2. 战斗系统:                                                          │
│     - 招数精妙: 不靠力量碾压，靠技巧和智慧                             │
│     - 战斗逻辑: 每场战斗都有铺垫、经过、高潮                           │
│     - 越级可能: 实力不是绝对，智慧和机遇同样重要                       │
│                                                                         │
│  3. 节奏把控:                                                          │
│     - 张弛有度: 战斗和日常交叉，节奏感好                               │
│     - 爽点均匀: 不是一味堆砌爽点，而是合理分布                         │
│     - 情感点缀: 亲情、爱情、兄弟情都有，但不喧宾夺主                   │
│                                                                         │
│  4. 升级设计:                                                          │
│     - 目标明确: 读者永远知道主角下一个目标是什么                       │
│     - 难度递进: 每个阶段的敌人都比上阶段强                             │
│     - 奖励丰富: 突破后获得新能力，奖励感十足                           │
│                                                                         │
└────────────────────────────────────────────────────────────────────────┘
```

**写作技巧拆解**:

| 技巧类型 | 描述                           | 示例                   |
| -------- | ------------------------------ | ---------------------- |
| 流畅叙事 | 长句连接，阅读体验佳           | 秦羽的修炼之路一气呵成 |
| 战斗分级 | 用具体境界区分实力             | 凡人→金丹→元婴→渡劫    |
| 机缘设定 | 奇遇是升级的关键               | 流星泪、云雾仙踪       |
| 兄弟义气 | 主角有生死之交                 | 侯费和小黑的兄弟情     |
| 意境描写 | 山水、星空、剑意等意境描写出色 | 星辰变中的宇宙描写     |

**风格样本示例**:

```原文
"我要成为强者。"

秦羽站在悬崖边，看着云海翻涌。

从八岁那年目睹父亲的战斗开始，这颗种子就在他心里生根发芽。

"流星泪会帮助你的。"

神秘的声音在他脑海中响起。

秦羽不知道这是谁的馈赠，但他知道，这是他改变命运的契机。

十年。

他用十年时间，将外功修炼到极致。

然后，他踏入了一条从未有人走过的路——暗星修炼之法。
```

**适用场景**:

| 场景类型 | 适用度 | 说明                       |
| -------- | ------ | -------------------------- |

## 错误处理

| 错误码 | 说明       | 处理方式                   |
| ------ | ---------- | -------------------------- |
| SL-001 | 作家不存在 | 使用默认风格，提供作家列表 |
| SL-002 | 样本不足   | 提示增加样本(≥5000字)      |
| SL-003 | 分析失败   | 使用通用风格，保留原文     |
| SL-004 | 风格冲突   | 提示选择单一风格参考       |
| SL-005 | 参数缺失   | 提示补充必要参数           |

**错误恢复流程**:

```
SL-001:
  - 输出所有可用作家列表
  - 引导用户选择

SL-002:
  - 计算当前样本字数
  - 提示需要补充的字数
  - 建议补充场景类型

SL-003:
  - 记录分析失败原因
  - 使用通用网文风格作为默认
  - 允许用户手动调整

SL-004:
  - 检测到多个风格冲突
  - 建议选择主要参考风格
  - 可以提供分场景选择

SL-005:
  - 列出必需参数
  - 提供使用示例
```
