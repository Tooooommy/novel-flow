# 内容生成技能

## 使用场景

- 用户要生成章节正文
- 用户要创作特定场景
- 用户要编写对话
- 用户要生成动作/战斗场景
- 用户要生成情感场景

---

## 命令

### 生成章节正文

> 依据小说大纲系统生成章节正文，考虑风格、节奏、场景、对话、描写、动作、情感等因素。

```
/nf-content-generator
```

**执行流程**:

1. style (风格应用)：根据选定风格获取写作指导
2. pacing (节奏设计)：确定本章节奏类型：铺垫/冲突/高潮/转折
3. scene (生成场景)：根据大纲和节奏生成场景布置
4. dialogue (生成对话)：生成角色对话（含潜台词）
5. description (生成描写)：生成环境、人物、氛围描写
6. action (生成动作)：生成动作描写、战斗动作
7. emotion (生成情感)：生成情感渲染、内心独白
8. 整合输出：组合各部分输出完整章节: `ch-{章节编号}-{章节名}.md`

**执行方案**:

| 步骤          | 操作                       | 产出     |
| ------------- | -------------------------- | -------- |
| 1.style       | 获取风格指导，应用写作特征 | 风格参数 |
| 2.pacing      | 确定本章节奏类型和目标     | 节奏计划 |
| 3.scene       | 生成场景布置、背景、道具   | 场景要素 |
| 4.dialogue    | 生成角色对话（含潜台词）   | 对话内容 |
| 5.description | 生成环境、人物、氛围描写   | 描写片段 |
| 6.action      | 生成动作描写、战斗动作     | 动作片段 |
| 7.emotion     | 生成情感渲染、内心独白     | 情感片段 |
| 8.整合输出    | 组合各部分输出完整章节     | 完整章节 |

**智能行为**:

- 自动衔接上章结尾
- 自动去AI味道
- 节奏控制

**章节字数标准**:

| 章节类型 | 推荐字数 | 允许范围  | 组成比例                |
| -------- | -------- | --------- | ----------------------- |
| 标准章   | 3000     | 2800-3200 | 场景60%+对话25%+其他15% |
| 过渡章   | 2500     | 2200-2800 | 场景55%+对话30%+其他15% |
| 高潮章   | 3500     | 3200-4000 | 场景65%+对话20%+其他15% |
| 决战章   | 4000+    | 3500-5000 | 战斗60%+对话25%+其他15% |

### pacing - 节奏设计

**执行方案**:

| 步骤 | 操作                                | 产出     |
| ---- | ----------------------------------- | -------- |
| 1    | 分析本章在大纲中的位置              | 章节类型 |
| 2    | 确定节奏目标（铺垫/冲突/高潮/转折） | 节奏类型 |
| 3    | 规划爽点出现位置                    | 爽点计划 |
| 4    | 生成节奏曲线                        | 节奏数据 |

**节奏类型定义**:

| 类型 | 说明                 | 适用场景 | 情感曲线特征     |
| ---- | -------------------- | -------- | ---------------- |
| 铺垫 | 引入新元素、埋下伏笔 | 章初     | 1.0→2.0 缓慢上升 |
| 冲突 | 矛盾升级、压力增大   | 章中     | 2.0→3.5 阶梯上升 |
| 高潮 | 爽点爆发、情节巅峰   | 章中后   | 3.5→5.0 急剧攀升 |
| 转折 | 意外发生、局势逆转   | 章末     | 5.0→2.0 骤然下降 |
| 收尾 | 解决当前、承接下章   | 章末     | 2.0→1.5 小幅回落 |

**节奏曲线示例**:

```
情感强度
    │
 5.0 │                              ★ 高潮
    │           ★ 高潮              (逆转)
 4.0 │      ★ 高潮    /           /
    │   ★ 高潮    / ★ 转折       /
 3.0 │      ↗ ↘  /            ★ 铺垫
    │   ★ 铺垫  /
 2.0 │
    │★ 铺垫
 1.0 │
    └──────────────────────────────→ 章节进度
      10%  20%  30%  40%  50%  60%  70%  80%  90%
```

**节奏量化模型**:

情感曲线计算:

- 基础值: 极度负面(-1.0) → 极度正面(+1.0)
- 情感词权重: 强1.5x / 中1.0x / 弱0.5x
- 转折词(但是/然而): 方向反转×0.8
- 递进词(更/尤其): 当前值×1.2

爽点密度公式: `S = (Σ爽点权重) / 字数 × 1000`

| 爽点类型 | 权重 | 示例                         |
| -------- | ---- | ---------------------------- |
| 逆转爽点 | 1.5  | 反杀、打脸、没想到、绝地翻盘 |
| 升级爽点 | 1.3  | 突破、晋升、觉醒、获得传承   |
| 情报爽点 | 1.2  | 原来、真相、秘密揭露         |
| 装逼爽点 | 1.4  | 实力打脸、嘲讽反击、众人震惊 |
| 情感爽点 | 1.1  | 佳人倒贴、兄弟情深、师徒情   |
| 收获爽点 | 1.0  | 获得宝物、习得技能、结识强者 |

张力指数: `T = 0.3×冲突 + 0.25×悬念 + 0.2×时间压力 + 0.15×风险度 + 0.1×不确定性`

**速度等级**:

| 等级 | 字数/场景 | 短句比例 | 内心描写 | 适用节奏   |
| ---- | --------- | -------- | -------- | ---------- |
| 极快 | 500-800   | ≥70%     | ≤10%     | 战斗、危机 |
| 快   | 800-1200  | 50-70%   | 10-20%   | 冲突、打斗 |
| 中   | 1200-2000 | 30-50%   | 20-30%   | 日常、过渡 |
| 慢   | 2000-3000 | 20-30%   | 30-50%   | 情感、心理 |
| 极慢 | >3000     | <20%     | >50%     | 沉思、回忆 |

**章节结构模板**:

| 结构位置 | 内容要求          | 字数占比 | 节奏控制 |
| -------- | ----------------- | -------- | -------- |
| 章首     | 场景建立+上章承接 | 10%      | 平稳引入 |
| 章初     | 情节推进+小冲突   | 20%      | 逐步升温 |
| 章中     | 矛盾升级+核心内容 | 30%      | 保持张力 |
| 章后     | 高潮爆发/转折     | 20%      | 急速攀升 |
| 章尾     | 悬念/收尾/钩子    | 20%      | 戛然而止 |

### scene - 生成场景

**执行方案**:

| 步骤 | 操作                                | 产出     |
| ---- | ----------------------------------- | -------- |
| 1    | 确定场景类型（室内/室外/战斗/日常） | 场景类型 |
| 2    | 根据背景设定生成场景细节            | 场景布置 |
| 3    | 添加道具、氛围元素                  | 场景道具 |
| 4    | 规划场景与情节的对应关系            | 场景节奏 |

**场景要素**:

```
场景: 宗门大殿
     │
     ├──→ 物理要素
     │      - 建筑: 大殿、柱子、匾额
     │      - 光线: 昏暗/明亮
     │      - 空间: 宽敞/狭窄
     │
     ├──→ 氛围要素
     │      - 气味: 檀香/血腥
     │      - 声音: 钟鸣/寂静
     │      - 温度: 阴冷/炎热
     │
     └──→ 情节要素
            - 关键道具: 令牌/书信
            - 关键位置: 主席/侧席
```

**场景类型标准**:

| 场景类型 | 特征要素           | 描写重点       | 常用技巧 |
| -------- | ------------------ | -------------- | -------- |
| 室内静态 | 封闭空间、固定物品 | 细节、氛围     | 定点观察 |
| 室内动态 | 多人活动、事件发生 | 人物互动、动作 | 镜头切换 |
| 室外自然 | 山川河流、季节变化 | 壮阔、意境     | 远近结合 |
| 室外社会 | 城市、集市、人群   | 烟火气、热闹   | 群像扫描 |
| 战斗场景 | 对抗、灵力、招式   | 动作、效果     | 高速镜头 |
| 回忆场景 | 模糊、褪色、情感   | 情感、心境     | 意识流   |

**场景描写标准**:

| 要素     | 字数占比 | 描写原则          |
| -------- | -------- | ----------------- |
| 空间设定 | 30%      | 用3-5个感官词建立 |
| 细节道具 | 30%      | 与情节相关才描写  |
| 氛围营造 | 20%      | 映衬情绪和主题    |
| 动态变化 | 20%      | 随情节推进演变    |

**场景五感描写标准**:

```
视觉（80%场景使用）:
  - 色彩: 明亮/暗淡/鲜艳
  - 形态: 形状/大小/动态
  - 光影: 明暗/投影/反射

听觉（60%场景使用）:
  - 声音类型: 人声/自然/机械
  - 音量: 轰鸣/低语/寂静
  - 节奏: 急促/平缓/断续

嗅觉（30%场景使用）:
  - 气味: 花香/血腥/檀香
  - 浓度: 浓郁/清淡/刺鼻

触觉（25%场景使用）:
  - 质感: 粗糙/光滑/冰凉
  - 温度: 炎热/寒冷/适宜

味觉（10%场景使用）:
  - 口中味道: 苦涩/甘甜
  - 场景关联: 餐饮/毒药
```

### dialogue - 生成对话

**执行方案**:

| 步骤 | 操作                                | 产出     |
| ---- | ----------------------------------- | -------- |
| 1    | 分析角色性格和当前情绪              | 角色状态 |
| 2    | 确定对话目的（试探/威胁/求情/隐瞒） | 对话目的 |
| 3    | 生成符合角色的台词                  | 台词内容 |
| 4    | 添加潜台词和言外之意                | 潜台词   |

**对话生成方法**:

```
输入: 主角(傲娇) vs 反派(阴险)
     │
     ├──→ Step 1: 角色分析
     │      - 主角: 外冷内热、不服输
     │      - 反派: 城府深、喜欢算计
     │
     ├──→ Step 2: 对话目的
     │      - 反派: 试探主角实力
     │      - 主角: 隐藏实力、麻痹对方
     │
     ├──→ Step 3: 台词生成
     │      - 反派: "听闻少侠天赋异禀，今日一见，果然名不虚传"
     │      - 主角: "哪里哪里，在下不过是运气好罢了" (隐藏实力)
     │
     └──→ Step 4: 潜台词
            - 反派: (你在装傻，我会查清楚的)
            - 主角: (你想套我的话，没门)
```

**对话类型标准**:

| 对话类型 | 特征       | 句式特点          | 字数标准        |
| -------- | ---------- | ----------------- | --------------- |
| 试探性   | 委婉、试探 | 疑问句/反问句为主 | 20字内          |
| 威胁性   | 强硬、施压 | 祈使句为主        | 15字内          |
| 解释性   | 说明、讲理 | 陈述句为主        | 40字内          |
| 情感性   | 表达情感   | 感叹句/短句       | 25字内          |
| 阴谋性   | 话中有话   | 双重含义          | 表面20字+潜台词 |

**对话生成模板**:

| 模板名称 | 适用场景 | 模板内容             |
| -------- | -------- | -------------------- |
| 三明治法 | 正常交流 | 开场白+核心内容+收尾 |
| 针锋相对 | 冲突对话 | 反击+质问+逼迫       |
| 欲擒故纵 | 心机对话 | 退让+诱导+陷阱       |
| 单刀直入 | 紧急场景 | 直奔主题+简短有力    |
| 迂回包抄 | 试探场景 | 旁敲侧击+逐渐深入    |

**潜台词设计标准**:

```
一级潜台词（直白型）:
  表面: "你很好"
  潜意: "你太好了，我配不上" (自卑)

二级潜台词（暗示型）:
  表面: "天气真不错"
  潜意: "我不想继续这个话题"

三级潜台词（反转型）:
  表面: "我很感激你"
  潜意: "你害了我，还敢来邀功？"

四级潜台词（多层型）:
  表面: "保重"
  潜意: "可能再也见不到你了" + "你要小心" + "我会想你"
```

**对话比例标准**:

| 章节类型 | 对话占比 | 对话密度 | 独白比例 |
| -------- | -------- | -------- | -------- |
| 日常章   | 35-40%   | 高       | 10-15%   |
| 战斗章   | 15-25%   | 低       | 5-10%    |
| 情感章   | 30-35%   | 中       | 15-20%   |
| 悬疑章   | 25-30%   | 中       | 10-15%   |

### description - 生成描写

**执行方案**:

| 步骤 | 操作                           | 产出     |
| ---- | ------------------------------ | -------- |
| 1    | 确定描写对象和类型             | 描写计划 |
| 2    | 调用五感描写（视/听/嗅/触/味） | 感官素材 |
| 3    | 添加修辞手法                   | 文学效果 |
| 4    | 整合成连贯段落                 | 描写片段 |

**描写类型**:

| 类型 | 说明             | 技巧     | 字数标准  |
| ---- | ---------------- | -------- | --------- |
| 环境 | 场景、氛围、天气 | 动静结合 | 200-400字 |
| 人物 | 外貌、表情、动作 | 白描为主 | 100-200字 |
| 心理 | 内心、情绪、思考 | 具象化   | 150-300字 |
| 动作 | 行为、手势、姿态 | 精准动词 | 100-300字 |

**人物外貌描写标准**:

| 要素 | 必写 | 选写 | 禁用             |
| ---- | ---- | ---- | ---------------- |
| 眼睛 | ★    | -    | 过于华丽的形容词 |
| 身形 | ★    | -    | 堆砌形容         |
| 气质 | -    | ★    | 抽象描述         |
| 服饰 | -    | ★    | 过多细节         |
| 发色 | -    | ★    | 非必要           |

**环境描写标准**:

| 描写层次 | 内容               | 字数      | 作用     |
| -------- | ------------------ | --------- | -------- |
| 远景     | 整体轮廓、氛围     | 50-100字  | 建立场景 |
| 中景     | 主体建筑、主要物体 | 100-200字 | 聚焦视线 |
| 近景     | 细节道具、关键物品 | 50-100字  | 推动情节 |
| 特写     | 微小细节、情绪对应 | 30-50字   | 渲染情绪 |

**白描与细描标准**:

```
白描（推荐80%场景）:
  "他推开门，走了进来。"
  特点: 简洁、动词为主、快速推进

细描（用于重要时刻20%场景）:
  "他缓缓推开那扇斑驳的木门，
   铰链发出吱呀的响声，
   像是在诉说着岁月的痕迹。"
  特点: 修饰词、多角度、渲染氛围
```

### action - 生成动作

**执行方案**:

| 步骤 | 操作               | 产出     |
| ---- | ------------------ | -------- |
| 1    | 确定事件类型和强度 | 事件级别 |
| 2    | 规划动作序列       | 动作列表 |
| 3    | 添加战术/策略要素  | 战术设计 |
| 4    | 生成动作描写       | 动作片段 |

**战斗强度分级**:

| 强度 | 描写密度 | 动作数量    | 时长 | 适用场景   |
| ---- | -------- | ----------- | ---- | ---------- |
| 1    | 低       | 3-5个动作   | 1段  | 偷袭、秒杀 |
| 2    | 中       | 5-10个动作  | 2段  | 普通对决   |
| 3    | 高       | 10-20个动作 | 3段+ | 高手过招   |
| 4    | 决战     | 20+动作     | 全章 | 终极对决   |

**动作描写模板**:

| 强度   | 模板结构                 | 示例                                                                                                              |
| ------ | ------------------------ | ----------------------------------------------------------------------------------------------------------------- |
| 低强度 | 结果+简单动作            | "李尘一剑刺出，张伟躲避不及，被剑气所伤。"                                                                        |
| 中强度 | 攻防转换+多个动作        | "张伟侧身躲过剑锋，反手一掌拍向李尘胸口。李尘收剑格挡，却被震得后退三步。"                                        |
| 高强度 | 招式名+动作细节+环境互动 | "李尘双目圆睁，体内灵力狂涌。'破天三式——'剑光暴涨，化作三道银虹，分别刺向张伟三处要害。张伟面色剧变，连连后退..." |

**战斗动作标准**:

| 要素     | 要求               | 示例                                  |
| -------- | ------------------ | ------------------------------------- |

| 动作连贯 | 一气呵成的动作流   | "收剑→侧身→反手刺出"                  |

| 灵力运用 | 玄幻特色           | 灵力流转、经脉运行                    |

### emotion - 生成情感

**执行方案**:

| 步骤 | 操作               | 产出     |

**情绪表现方法**:

| 情绪 | 身体反应                     | 内心独白           |
| ---- | ---------------------------- | ------------------ |
| 愤怒 | 握拳、咬牙、脸红、发抖       | "我要让你付出代价" |
| 悲伤 | 垂眼、叹息、颤抖、哽咽       | "为什么偏偏是我"   |
| 喜悦 | 嘴角上扬、眼中有光、轻快步伐 | "终于等到这一天"   |
| 恐惧 | 后退、冷汗、僵硬、瞳孔放大   | "这不可能..."      |
| 惊讶 | 瞪大眼、倒吸凉气、身体僵住   | "竟然是这样"       |
| 厌恶 | 皱眉、后退、语气冷淡         | "虚伪"             |
| 爱慕 | 脸红、心跳加速、目光躲闪     | "他/她..."         |

**情感渲染技巧**:

| 技巧            | 方法                   | 示例                                         |
| --------------- | ---------------------- | -------------------------------------------- |
| Show Don't Tell | 用动作展示，不直接陈述 | ✗"他非常愤怒" ✓"他握紧拳头，指节发白"        |
| 环境映衬        | 用环境映衬情绪         | "窗外雷声轰鸣，映衬着他此刻的心情"           |
| 内心独白        | 揭示深层情绪           | "（表面上平静）内心：她怎么能这样对我..."    |
| 对比反衬        | 用对比强化情绪         | "锣鼓喧天，人声鼎沸，而他只觉得世界一片寂静" |
| 细节放大        | 放大情绪相关的微小细节 | "她注意到他的手指微微颤抖，原来他也紧张"     |

**内心独白标准**:

| 类型     | 字数     | 使用场景 | 示例                              |
| -------- | -------- | -------- | --------------------------------- |
| 即时反应 | 10字内   | 紧急时刻 | "糟了！"                          |
| 思考分析 | 30-50字  | 决策时刻 | "如果这样...那么..."              |
| 情感宣泄 | 20-40字  | 情感爆发 | "凭什么！明明是我的..."           |
| 回忆联想 | 50-100字 | 情感触发 | "看着眼前的她，他想起了十年前..." |

**情感层次标准**:

```
单一情绪:
  "他很高兴。"

多层情绪（推荐）:
  "他笑了笑，但眼底却没有笑意。"
  外在表现 + 内在真实

情绪转变:
  平静表面 → 内心波动 → 情绪爆发 → 强行压制 → 表面平静
```

## 技巧总结

**快速生成技巧**:

1. **先节奏后内容**: 先确定本章节奏曲线，再填充内容
2. **场景五感法**: 每个场景至少调动2种感官
3. **对话潜台词**: 每段重要对话至少有一层潜台词
4. **动作精准词**: 动词要具体，避免"做"、"搞"
5. **情绪身体现**: 情绪通过身体反应展现，不直接说

**质量提升技巧**:

1. **首尾呼应**: 章首埋下伏笔，章尾呼应
2. **动静结合**: 场景描写动静交替
3. **多视角转换**: 适时转换视角增加层次
4. **留白艺术**: 重要时刻不写满，留给读者想象
5. **五感交织**: 视觉、听觉、嗅觉、触觉、味觉综合
