## 快速开始指南

### 1. 创建新 Skill

```bash
# 复制模板
cp .trae/templates/SKILL_TEMPLATE.md .trae/skills/nf-[模块名].md

# 编辑文件，替换所有占位符
```

### 2. 必须包含的命令

| 命令类型 | 说明 | 何时添加 |
|---------|------|---------|
| `full` | 一键执行所有子命令 | 必须 |
| `list` | 列出可用资源 | 可选 |
| `get` | 获取单个资源 | 可选 |
| `apply` | 应用配置/规则 | 可选 |

### 3. 命令命名规范

- 使用小写字母
- 使用连字符分隔单词
- 必须包含 `full` 命令
- 避免使用与系统命令冲突的名称
