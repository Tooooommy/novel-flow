# 主线剧情

## 整体脉络

```
穿越星际 → 发现商机 → 小本经营 → 初具规模 → 遭遇危机 → 绝地反击
    ↓
扩张分店 → 商业竞争 → 势力崛起 → 星际闻名 → 商会打压 → 绝处逢生
    ↓
星际大战 → 帝国覆灭 → 立场对立 → 最终对决 → 奶茶帝国称霸宇宙
```

## 三段式结构

### 卷1：起步（地球→星际边缘）
- **起因**：程序员加班猝死，穿越到星际时代
- **发展**：发现奶茶市场空白，开始小本经营
- **高潮**：遭遇银河商会打压，绝地反击
- **结局**：站稳脚跟，赚得第一桶金

### 卷2：扩张（建立奶茶帝国）
- **起因**：不甘平凡，决定扩张
- **发展**：连锁店模式，进入核心星系
- **高潮**：与星际餐厅联盟的商业战争
- **结局**：奶茶帝国初具规模

### 卷3：征服（星际大战+终极对决）
- **起因**：星际大战爆发，商业格局巨变
- **发展**：主角被迫选边站，遭遇生存危机
- **高潮**：帝国覆灭危机，主角绝地翻盘
- **结局**：击败最终BOSS，奶茶帝国称霸宇宙
