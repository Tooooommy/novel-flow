# 《星际卖奶茶》总览

## 基本信息

| 项目 | 内容 |
|------|------|
| 题材 | 科幻星际 |
| 风格 | 轻松热血 |
| 主线 | 创业+星际征服 |
| 核心卖点 | 奶茶创业+星际战争+热血逆袭 |

## 核心设定

### 金手指
- 地球饮品配方（奶茶、果汁、咖啡等）
- 星际贸易系统（可升级）

### 主角设定
- 身份：地球程序员穿越者
- 性格：聪明、勤俭、有点腹黑
- 目标：用奶茶征服宇宙

### 世界观
- 星际联邦时代，地球为边缘星球
- 各星球种族林立，商业发达
- 奶茶为稀有商品，主角垄断技术

## 分卷大纲

| 卷 | 名称 | 核心剧情 | 目标字数 |
|----|------|----------|----------|
| 卷1 | 起步 | 地球→星际边缘 | 30万字 |
| 卷2 | 扩张 | 建立奶茶帝国 | 35万字 |
| 卷3 | 征服 | 星际大战+终极对决 | 35万字 |

## 主线节奏

```
卷1: 发现商机 → 小本经营 → 初具规模 → 遭遇危机 → 解决危机
卷2: 扩张分店 → 商业竞争 → 势力崛起 → 星际闻名 → 卷2终
卷3: 星际大战 → 帝国覆灭 → 最终对决 → 奶茶帝国称霸宇宙
```
