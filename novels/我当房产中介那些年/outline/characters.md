# 人物设定

## 主角

### 陈默

| 项目 | 内容                                      |
| ---- | ----------------------------------------- |
| 姓名 | 陈默                                      |
| 性别 | 男                                        |
| 年龄 | 25岁                                      |
| 职业 | 房产中介（成长路线：新人→精英→店长→老板） |

**身份设定**：

- 表面身份：普通大学毕业，失业青年，房产中介新人
- 真实身份：房源系统宿主（隐藏身份，初期不揭示）
- 最终身份：房产界传奇，诚信经营的代表

**性格特征**：

- 核心性格：外表温和、内心坚定
- 性格短板：有时候过于正直，不懂变通
- 性格发展：从小心翼翼→敢于挑战→成熟稳重

**外貌特征**：

- 整体印象：长相普通但眼神锐利
- 标志性特征：观察时会微微眯眼
- 习惯动作：思考时会摸后脑勺

**金手指**：

- 类型：系统流（房源系统）
- 名称：房源系统
- 能力：探测房源隐藏属性、评估投资价值、洞察命运
- 成长性：随成交经验升级，解锁更多功能
- 代价/限制：不能直接透露系统存在，否则系统会消失

**目标体系**：

- 短期目标（第1-50章）：在链家站稳脚跟，赚到第一桶金
- 中期目标（第50-200章）：成为门店销冠，开设自己门店
- 长期目标（第200章+）：改变行业规则，成为行业标杆

**成长弧**：

- 起点：失业青年，生活窘迫
- 触发点：入职链家，获得房源系统
- 第一阶段（第1-100章）：快速成长期，积累经验和口碑
- 转折点（第150章）：遭遇行业黑幕，被打压
- 第二阶段（第150-400章）：沉淀反击，建立自己势力
- 结局：实现目标，改变行业

## 女主

### 小雪

| 项目 | 内容                       |
| ---- | -------------------------- |
| 姓名 | 小雪                       |
| 性别 | 女                         |
| 年龄 | 24岁                       |
| 职业 | 房产中介（同事）           |
| 身份 | 链家门店同事，后期成为女主 |

**性格特征**：

- 核心性格：甜美可人、善解人意
- 隐藏性格：外柔内刚、有自己的坚持
- 与主角关系：初期同事，后期恋人

**人物定位**：

- 表面定位：门店甜美担当，负责维护女客户
- 实际定位：与主角志同道合，共同的理想伙伴

**情感线**：
| 阶段 | 章节 | 事件 |
|------|------|------|
| 相遇 | 第5章 | 同时入职链家 |
| 好感 | 第30章 | 共同完成任务 |
| 暧昧 | 第80章 | 工作配合默契 |
| 误解 | 第180章 | 误会主角动机 |
| 明确 | 第250章 | 关键事件表白 |
| 圆满 | 第500章 | 确定关系 |

**结局走向**：

- 与陈默最终确定关系
- 成为陈默事业的重要伙伴
- 实现职业成长，成为房产精英
