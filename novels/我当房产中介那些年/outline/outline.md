## 主线剧情概述

失业青年陈默入职房产中介，意外获得"房源系统"，能看到每个房源的特殊属性。从一个小中介到行业大佬，房产背后的故事比想象中精彩得多。

### 核心卖点

1. **房源系统**：能看到房源隐藏属性（凶宅、发生过重大事件、业主真实处境）
2. **行业揭秘**：房产中介行业的潜规则、黑幕、套路
3. **逆袭成长**：从小员工到行业大佬的奋斗历程
4. **商战博弈**：中介之间的竞争与合作

## 金手指体系

### 房源系统

| 功能     | 等级要求 | 效果                       |
| -------- | -------- | -------------------------- |
| 基础探测 | 初始     | 显示房源基本信息           |
| 隐藏属性 | Lv5      | 显示是否凶宅、发生过的事件 |
| 价值评估 | Lv10     | 评估投资价值和增值潜力     |
| 命运洞察 | Lv20     | 显示与房源相关人员的命运线 |
| 全知领域 | Lv30     | 区域所有房源信息一览       |

### 升级方式

- 成交房源数量积累经验值
- 客户满意度影响经验加成
- 特殊房源（凶宅、豪宅）额外经验
