# 世界观设定

## 都市房产中介行业

### 主要平台/公司

| 平台       | 定位     | 特点                       |
| ---------- | -------- | -------------------------- |
| 链家       | 高端直营 | 真实房源、服务费高、口碑好 |
| 贝壳       | 平台模式 | 整合多家中介、ACN合作      |
| 我爱我家   | 中端市场 | 门店多、覆盖面广           |
| 中原地产   | 港资背景 | 高端豪宅为主               |
| 一些小中介 | 灵活经营 | 套路多、风险大             |

### 行业规则

#### 收费规则

| 类型       | 收费标准       | 备注         |
| ---------- | -------------- | ------------ |
| 中介费     | 总价2-3%       | 买卖双方各付 |
| 贷款服务费 | 贷款额1-2%     | 可协商       |
| 过户费     | 固定额度       | 各地不同     |
| 独家委托   | 额外一个月租金 | 房东付费     |

#### 行业潜规则

| 规则     | 说明               | 主角态度             |
| -------- | ------------------ | -------------------- |
| 虚假房源 | 用低价房源吸引客户 | 抵制                 |
| 飞单     | 把客户资源转到别处 | 抵制                 |
| 抬价杀价 | 故意抬高房价再杀价 | 巧妙利用             |
| 阴阳合同 | 报低成交价逃税     | 抵制                 |
| 虚假成交 | 刷业绩数据         | 初期被迫参与，后抵制 |

### 客户类型

| 类型     | 特征     | 需求      | 应对策略     |
| -------- | -------- | --------- | ------------ |
| 刚需购房 | 首次购房 | 价格敏感  | 提供实用建议 |
| 改善型   | 换房需求 | 品质要求  | 精准匹配     |
| 投资客   | 多套持有 | 增值潜力  | 提供投资分析 |
| 炒房团   | 短进短出 | 快进快出  | 警惕风险     |
| 拆迁户   | 突然有钱 | 挥霍/理财 | 理性建议     |
| 婚房需求 | 结婚购房 | 稳定安全  | 提供安全感   |

### 房源类型

| 类型       | 特征       | 市场特点     |
| ---------- | ---------- | ------------ |
| 商品房     | 正常交易   | 最大份额     |
| 动迁房     | 限制交易   | 价格低       |
| 经济适用房 | 政府补贴   | 限制多       |
| 凶宅       | 发生过死亡 | 价格低但难卖 |
| 抵押房     | 业主欠债   | 交易风险大   |
| 拍卖房     | 法院执行   | 价格低但复杂 |

## 城市背景设定

### 主要场景

| 场景         | 类型       | 用途                 |
| ------------ | ---------- | -------------------- |
| 链家门店     | 工作场景   | 日常办公、开会、培训 |
| 各大小区     | 带看场景   | 核心业务场景         |
| 房产交易中心 | 交易场景   | 过户、缴税           |
| 高端写字楼   | 商业谈判   | 大客户、谈判         |
| 咖啡厅/餐厅  | 非正式交流 | 维护客户关系         |
| 主角住所     | 生活场景   | 休息、日常           |

| 层级     | 区域   | 房价      | 目标客户       |
| -------- | ------ | --------- | -------------- |
| 一线城市 | 核心区 | 10万+/㎡  | 豪宅、高端改善 |
| 一线城市 | 近郊   | 5-10万/㎡ | 改善刚需       |
| 二线城市 | 核心区 | 3-8万/㎡  | 改善刚需       |
| 二线城市 | 近郊   | 1-3万/㎡  | 刚需上车       |

## 系统设定

### 房源系统规则

| 规则     | 说明                                 |
| -------- | ------------------------------------ |
| 显示范围 | 以主角为中心的500米半径              |
| 信息类型 | 基础信息、隐藏属性、价值评估、命运线 |
| 更新频率 | 实时更新房源状态                     |
| 等级限制 | 高级功能需要等级解锁                 |
| 系统惩罚 | 恶意提供错误信息会被扣经验           |
| 系统奖励 | 成交优质房源额外奖励                 |

### 系统升级树

```
Lv1: 基础探测
  └→ Lv5: 隐藏属性
      └→ Lv10: 价值评估
          └→ Lv15: 客户画像
              └→ Lv20: 命运洞察
                  └→ Lv30: 全知领域
```

### 经验获取规则

| 行为         | 经验值 | 备注             |
| ------------ | ------ | ---------------- |
| 成交租房     | +100   | -                |
| 成交二手房   | +500   | 房价越高加成越多 |
| 成交豪宅     | +1000  | 额外奖励         |
| 发现凶宅     | +200   | 特殊奖励         |
| 客户好评     | +50/次 | -                |
| 成交问题房源 | -100   | 惩罚             |
| 飞单行为     | -500   | 严重惩罚         |
